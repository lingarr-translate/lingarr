# Translation services

Lingarr supports two kinds of translation services:

- **AI services** use an AI completion model to translate subtitles. They need an API key, a model choice, and optionally a custom prompt. See [AI services](/translation-services/ai-services).
- **Machine translation services** translate through a dedicated translation API and usually only need an API key. See [Machine translation](/translation-services/machine-translation).
